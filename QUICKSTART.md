## Troubleshooting

### Colors not red?
- Clear browser cache: `Ctrl+Shift+Delete`
- Restart dev server: `npm run dev`

### Text not updating?
- Check `src/i18n.js` for translation keys
- Refresh browser

### Images not showing?
- Check URLs in `src/components/Hero.jsx`
- Or see `IMAGE_ASSETS.md` to replace with local images
